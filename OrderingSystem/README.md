# Ordering System

## Overview

The Ordering System is a console-based Java application that allows users to place orders through a menu-driven interface. It enables users to select items, calculate the total bill, and display an order summary. The project demonstrates the implementation of Core Java concepts through a simple and interactive ordering process.

## Features

* Menu-driven console application
* Product selection using switch statements
* Order placement and processing
* Quantity-based bill calculation
* Total amount calculation
* Order summary generation
* User-friendly console interaction
* Modular program structure using methods

## Core Java Concepts Used

* Switch Statements
* Conditional Statements
* Loops
* Methods
* Scanner Class
* Variables and Data Types
* Arithmetic Operations
* Modular Programming

## Technologies Used

* Java
* Core Java
* Console-Based Application
