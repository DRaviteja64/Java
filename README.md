# Smart Parking Management System

## Overview

The Smart Parking Management System is a console-based Java application developed to automate apartment parking operations. It manages resident and visitor registrations, vehicle registration, parking slot allocation, monthly passes, visitor payments, vehicle search, parking exit, revenue reports, and parking slot availability through a menu-driven interface. The system is designed using Object-Oriented Programming (OOP) principles to provide a structured and efficient parking management solution.

## Features

* Supports multiple vehicle types:

  * Car
  * Bike
  * Electric Vehicle
* Resident and Visitor registration
* Vehicle registration with duplicate vehicle validation
* Parking slot allocation and management
* Vehicle search using registration number
* Vehicle entry and exit management
* Monthly pass issuance and cancellation for residents
* Visitor parking payment management
* Revenue report generation
* Parking slot occupancy display
* Resident and Visitor information display
* Menu-driven console application

## Object-Oriented Programming Concepts Used

* Encapsulation (Private fields with getters and setters)
* Inheritance
* Interfaces
* Runtime Polymorphism (Dynamic Method Dispatch)
* Method Overriding
* HAS-A Relationship (Object Composition)

## Technologies Used

* Java
* Core Java
* Object-Oriented Programming (OOP)
* Collections (`HashMap`)

* Console-Based Application
